#### Normar de uso

Recuerda que es necesario tener instalado NodeJS y decasrgar los repositorios:

  

- npm install

  

El ejercicio de ejecuta en el archivo app.js de la carpeta:`src`

  

1.  `cd src`

2.  `node app`
